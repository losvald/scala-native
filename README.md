# MiniScala(2) ports of [Scala Native's](http://www.scala-native.org/) benchmarks

- MiniScala supports a *subset* of Scala's syntax (`*[^2].scala` files)
- MiniScala2 supports a superset of MiniScala's syntax (`*2.scala` files)
